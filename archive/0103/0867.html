Nothing personal (to anyone), but I'm getting rather tired of this debate.

For the record:

1) I don't personally think AI's will mind which pronouns we use, since our
language does not provide ones that exactly fit the situation.

2) I believe that the cleanest and easiest way to proceed is to use the
English language that everyone knows and understands.

3) For my own writing, I will be using English. If I use the 'it' set or
the 'she' set has not yet been decided.

James Higgins
