James, when we say human equivalent we don't mean performance, we mean algorithms. Doing the thinking using the same algorithms that a human uses. We assume we can figure out these algorithms because we have an example to study. These human equivalent algorithms will run faster and more effectively on a silicon substrate than a biological one.
