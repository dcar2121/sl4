Hi all,

Not sure if any of you have seen the BBC series Look Around You, but
there is a particular episode which features AI Boxing. The second
series show entitled "Computers" (
http://www.bbc.co.uk/comedy/lookaroundyou/programmes/computers/ )
features a super computer called Bournemouth that they literally place
in an Iron cage and then ask it to escape before the end of the show.

I should warn you however that "Look Around You" is a spoof tv series
set in the 80s that looks at inventions of the future. So more for
entertainments sake then anything else.

Couldn't find the appropriate clip on Youtube (although the entire
first season is available there), but I'm sure you all have ways to
find such things ;)
