The biweekly times for unofficial chats are Saturday at 11AM and Wednesday at
9PM, EST.

The next official chat is Wednesday, March 6th. The time will be moved to 9PM
EST, so that it coincides with the biweekly time. (Previously the March chat
was scheduled for 8PM EST.)

Our monthly meetings will alternate between the first Wednesday for
odd-numbered months, and the first Saturday for even-numbered months, at the
corresponding times.

All of this is now part of the SL4 list description at:
http://sysopmind.com/sing/SL4.html#chat

Also, the page now explicitly allows you to search the SL4 archives via
Google. Long overdue, I know, but it's there at last.
