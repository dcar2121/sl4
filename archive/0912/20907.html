This is nice; a single phosphorus atom can act as a transistor.

http://www.tkk.fi/en/current_affairs/news/view/yhden_atomin_transistori_loydetty/

Of course, the gates in which it is set are a lot larger, but still getting closer to ultimate miniaturization. Atom-by-atom construction of logic circuits gets easier if you know that a main component only requires one atom.
