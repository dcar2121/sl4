> Schmidhuber spoke of proving only that the code modification had
> greater expected utility. Even if you proved the new system would do

the Hutters universal search algorithm. A very intuitive, but good
explanation can be found on this page:

http://fortnow.com/lance/complog/archive/2003_05_04_archive.html

(Look for universal search.)
