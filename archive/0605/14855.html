On Wed, 2006-05-10 at 21:33 -0500, Chris Capel wrote:
> Is friendliness affected by the amount of memory available? Could, for
> instance, adding more memory change a proved-Friendly AI into an
> Unfriendly one?

In principle, yes. Just write a program that checks to see if it has
more than 10^20 bits of memory, and destroys the universe if so, but
otherwise is Friendly.

In practice, I don't think this will be an issue.
