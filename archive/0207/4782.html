Rereading this, either you're joking/trolling, or you're a really
sucky expert system with a pretty decent (for the current art) and
cleverly camouflaged natural language system bolted on to it -- or you
*seriously* need to get some education so you can talk about these
things sensibly.

to not upgrade us/use all available matter:

TK> Singularity around, that is. I say it isn't.)

TK> We need some matter, of course. We don't need the part of the Sun
TK> shining away, the Earth interior ... and so on.

Remove the earth's interior? Remove the side of the sun facing away
from us?

using all caps or html formatting.
