Ben Goertzel wrote:

> the semantic
> markup is going to mainly be inserted by (initially primitive) AI's, not by
> humans.
>
> AI's that are costly to operate can read a Web page once and insert semantic
> markups.

If one has AIs that can usefully extract the appropriate semantics from
unstructured data, what's the benefit to having them then do structured markup?
If you've got the former, the latter seems like a performance hack at best, a
waste of bits and cycles at worst.

I.e., the semantic Web already exists. :-)
