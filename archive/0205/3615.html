> Does anyone have suggestions for a tool that does this, hopefully an
> inexpensive one? Failing that, does anyone know which search terms would
> describe this kind of tool? Feel free to answer privately.

You might try Scribe. It looks useful for building fairly complicated
document frameworks. It is a markup language (of a sort) though...

        http://www-sop.inria.fr/mimosa/fp/Scribe/

It supports full Scheme, so the relations you want to describe should
be fairly easily implemented as an extension to the base language.
