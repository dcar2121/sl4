The short answer is that a setup like this won't do anything nontrivial at
all, though if you want to code it up like I did and see for yourself by all
means go ahead. If you want to check prior art and save yourself some time,
two names that come to mind are Tierra, and some of Koza's work - not his
mainstream stuff that tries to optimize for a particular fitness function
(which you've probably already come across), but some early incidental
experiments he did with getting stuff to spontaneously emerge.
